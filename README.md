# Retirement Planner

A personalized retirement financial planning tool.

## Features
- Savings runway calculator
- Monthly budget vs. income breakdown
- Social Security + pension income planning
- Investment growth projections
- State-by-state tax & cost of living impact (all 50 states)

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Build for Production

```bash
npm run build
```

## Stack
- React 18 + Vite
- Recharts (charts)
- No backend required — fully client-side
